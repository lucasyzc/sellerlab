# 周度运营手册 v1（无数据库）

## 1) 目标

在没有数据库与 BI 的情况下，稳定跑出每周经营节奏。

## 2) 每周流程（固定）

1. 周一上午：从埋点日志或统计工具导出上周数据
2. 周一中午：填充 `weekly-ops-template-v1.csv`
3. 周二：30 分钟复盘会（只看 3 个问题）
4. 周三-周五：执行 1-2 个优化动作

## 3) 3 个必看问题

- 工具使用率是否上升？（`tool_used_uv / tool_page_uv`）
- 留资率是否上升？（`email_captured_count / tool_page_uv`）
- 最大流失发生在哪一步？（访问 -> 使用 -> CTA -> 留资）

## 4) 建议阈值（早期参考）

- `tool_use_rate` >= 15%
- `cta_click_rate` >= 8%
- `lead_capture_rate` >= 3%

若连续两周低于阈值，优先优化：

- 工具结果页 CTA 文案
- 表单字段数量（尽量少）
- 页面加载速度和移动端体验

## 5) 首批实验建议

- 实验 A：CTA 文案（效率导向 vs 收益导向）
- 实验 B：表单位置（工具上方 vs 结果区下方）
- 实验 C：中英文落地页主标题差异
